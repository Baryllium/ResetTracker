# ResetTracker

## Installation:

Follow [TalkingMime's Tutorial](https://youtu.be/KIAo3Lgsk_Q), but instead of giving the program your saves path you give it your SpeedrunIGT record folder (ie: C:\\Users\\NAME\\speedrunigt\\records)

## Required Mods:
- Atum 1.0.5+
- SpeedrunIGT 8.1+
